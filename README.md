 Marketing Conversion Analysis: Facebook vs Google AdWords
 Project Overview

This project presents a data-driven comparison between Facebook Ads and Google AdWords to determine which platform generates a higher number of conversions.

The analysis was developed as part of a data analytics capstone-style project and focuses on applying descriptive statistics, hypothesis testing, correlation analysis, and simple linear regression to support marketing decision-making.

The final outcome provides actionable insights to optimize advertising budget allocation and improve return on investment (ROI).

 Business Context

BrightThreads invests in paid digital advertising across multiple platforms, including Facebook and Google AdWords. Management aims to identify which platform performs better in terms of conversions in order to make informed decisions about future budget distribution.

Business Question:

Which advertising platform generates more conversions and should receive a higher budget allocation?

 Methodology

The analysis followed a structured data analytics workflow:

Data understanding and variable classification

Descriptive statistics (mean, median, mode, standard deviation)

Exploratory Data Analysis (EDA) using frequency tables and histograms

Correlation analysis between clicks and conversions

Hypothesis testing to compare conversion performance

Simple linear regression to model the relationship between variables
